Welcome To Atturaif!

Atturaif is An Interactive Tour Guide Application and uses this website to manage the Application activites, the app will provide an interactive avatar to help simulate the old life. Also, visitors will be able to view information about the artifacts in Atturaif museum using visual objects. The application will provide a booking feature to facilitate booking a tour or a lounge for visitors. In general, using augmented reality in touristic places in Saudi Arabia will attract tourists from different ages and interest. Also, it is going to be a delightful advertisement for Atturaif.
